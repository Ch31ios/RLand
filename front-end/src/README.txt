
ESTRUCTURA DEL PROYECTO:

src/  
├── assets/  
│   ├── images/: Imágenes (logotipos, iconos, fondos, etc.).  
│   └── styles/: Archivos CSS (globales y específicos de componentes).  
├── components/: Componentes reutilizables de la interfaz de usuario.  
├── pages/: Componentes que representan las diferentes páginas de la aplicación.
│
├── api/ (opcional): Servicios para interactuar con APIs externas.  
├── hooks/ (opcional): Custom hooks para lógica reutilizable.  
├── utils/ (opcional): Funciones auxiliares útiles en toda la aplicación.  
└── tests/ (opcional): Pruebas unitarias y de integración.
