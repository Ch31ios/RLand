
# Guía de Instalación del Proyecto
Antes de comenzar, asegúrate de tener instalados los siguientes requisitos:

- **Node.js** - https://nodejs.org/en/download/prebuilt-installer
- **npm** - (viene incluido con Node.js)

Para verificar que tienes ambas herramientas instaladas, puedes ejecutar los siguientes comandos en tu terminal:

```bash
node -v
npm -v
```

### Pasos para clonar y configurar el proyecto


1. Clona el repositorio en tu máquina local:

    ```bash
    git clone https://github.com/Ch31ios/RLand.git
    ```

2. Navega al directorio del proyecto:

    ```bash
    cd RLand
    ```

3. Accede a la carpeta del frontend:

    ```bash
    cd front-end
    ```

4. Instala todas las dependencias necesarias ejecutando el siguiente comando:

    ```bash
    npm install
    ```

5. Finalmente, inicia la aplicación con:

    ```bash
    npm start
    ```

Una vez realizados estos pasos, el proyecto estará configurado y listo para ejecutarse localmente.
